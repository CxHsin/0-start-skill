# 0-start-skill

`0-start` 是一个平台无关的 skill，用来帮助 AI 先从真实招聘语言和高质量一手资料里建立陌生领域的第一印象，再逐步进入答疑和学习路径。

它适合安装到任何支持自定义指令、skill、agent 或插件式配置的平台上。这个仓库只提供行为定义，不绑定固定 UI 或元数据格式。

## 适合什么场景

- 你想让 AI 先帮你看懂一个新领域，而不是直接给课程表
- 你希望 AI 先扒 JD，再给术语、资料和学习路线
- 你想把“冷启动教练”能力装进任何能加载说明文件的 AI 平台

## 安装指南

把这个仓库或其中的 `SKILL.md` 提供给你的 AI，然后直接发送下面这段话：

```text
请安装并遵循仓库里的 0-start skill。先读取 SKILL.md，理解它的目标、工作流和边界，然后在后续对话中按它的规则工作：先从 JD 风格信号和一手资料帮助我建立第一印象，再进行追问和解释，最后才给学习路径。
```

如果你的平台支持把文件作为上下文附件，直接附上 `SKILL.md` 效果最好。

## 它会怎么工作

安装后，`0-start` 会默认走这条路径：

1. 从用户给出的领域名开始
2. 提取 JD 风格的市场信号
3. 给出少量核心术语和一手资料入口
4. 在用户卡住时继续答疑
5. 等用户形成第一印象后再给学习路线

## 维护原则

- 先稳定，再扩展
- 一手资料优先，必要时才降级
- 只有在真的会改变输出时才追问
- 让新人先建立感觉，再谈路线
